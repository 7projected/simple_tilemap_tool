# simple_tilemap_tool
Very janky.

# Format
First 2 lines designated for the width and height of a map.
The big blob of hexadecimal values after that is a combination of all the tiles in a map.

# Controls
W-Move up
S-Move Down
A-Move left
D-Move right
LMB-Draw
RMB-Erase
F1-Save file
